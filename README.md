# 🚀 Quick Start

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/abdullahijaz5961/interviewai-pro.git
cd interviewai-pro
```

## 2️⃣ Launch Backend

```bash
cd backend

pip install -r requirements.txt

alembic upgrade head

uvicorn app.main:app --reload
```

Backend will start at:

```
http://127.0.0.1:8000
```

---

## 3️⃣ Launch Frontend

```bash
cd frontend

npm install

npm run dev
```

Frontend will start at:

```
http://localhost:5173
```
